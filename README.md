# Matdongsan
## - 데이터 크롤링 기반 부동산 데이터 제공 서비스 -
![header](https://capsule-render.vercel.app/api?type=transparent&height=100&text=나의%20역할%20:%20Java(%20jsp,%20servlet),%UI/UX디자인&fontColor=8A0A8A&fontSize=40)
#### ![programming-language](https://user-images.githubusercontent.com/107903374/207483181-39988ebf-7c6c-443b-a976-fae5fe9b07a2.png) JSP와 Spring, HTML/CSS/JS등을 사용해 웹 서비스의 UI/UX 설계 및 프론트엔드 기능 구현
 - 홈페이지 템플렛 전체 수정
 - JSP/Servlet을 이용한 Client와 Server Controller 연결
 - 마이 페이지 설계
 - 세부 css 조정
 - Bootstrap을 이용한 반응형 웹사이트 구현
 - Header/Footer jsp 파일 include 작업
 
 #### 📜 프로젝트를 진행하며 얻은 것
- 학습한 기술
  - GitHub를 활용해 버전 관리 및 협업하는 방법
  - JSP 파일에서 Servlet 파일로 데이터를 보내고 원하는 방식으로 데이터를 처리한 후, 다시 JSP 파일로 값을 보내는 법
  - JS,JQuery 등 스크립트를 사용한 기능 구현 방법 학습
  - Bootstrap과 CSS 등을 이용해 좀 더 Client 친화적인 웹서비스를 할 수 있는 사이트 구현
  - 많은 회의로 커뮤니케이션 기술 상승
  
  
- 어려웠던 부분, 오류 상황
  - 하루마다 프로젝트 전 후로 회의를 통해 팀원들과의 커뮤니케이션이 중요하다는 점
  - 부동산 계산기 페이지를 설계하고 기능을 구현하려 했지만 구글링의 한계와 생소한 단어 및 코드에 대한 분석 실패로 인해 구현하지 못 함 
    -> 다른 팀원이 다른 방식으로 구현
  - 관리자 페이지를 구현하지 못 함

### 프로젝트를 진행한 이유
![image](https://user-images.githubusercontent.com/107903374/207484046-5b6adc4f-4cc5-4b85-ab6f-11094b7099ff.png)
 - 순천 지역에서 실거주를 목적으로 매물을 찾는 청년들에게 경제 상황, 근무지, 선호 편의시설, 인프라 등을 고려하여 적절한 매물을 찾아 주고, 부동산 매입에 필요한 정보(대출 및 세금 정보)를 제공하기 위해 이 프로젝트를 진행
 - 이용자들은 부동산 시장을 파악할 수 있고 대출의 정도를 계산할 수 있으며 자신에게 맞는 부동산 거래를 할 수 있게 서비스

 ##### 개발 목표
 ![image](https://user-images.githubusercontent.com/107903374/207488573-831777b0-be95-4b84-b050-9d5d2af9502e.png)

 ##### 개발 내용
  - 회원가입, 로그인, 매물 관련 정보제공 등 기본적인 기능 제공
  - 맞춤 검색 서비스 : 다양한 요소로 매물을 검색할 수 있도록 자신의 조건에 알맞은 매몰 조회 서비스를 제공한다.
  - 주변 상권 정보제공 서비스 : 매물 주변의 상권 정보를 제공한다.
  - 실거래가와 대출 및 세금 정보제공 서비스 : 부동산 매입에 필요한
    정보인 실거래가와 대출 및 세금 정보를 제공한다.
  - 지도 제공 : 자신이 조회한 매물의 지도 정보를 확인할 수 있음
  - 대출 계산기 : LTV, DTI, DSR에 따른 예상 대출 금액 정보제공
  - 각종 부동산과 관련된 세금 정보제공

  - 요구사항 정의서
  - DB 요구사항 분석서
  - 테이블 명세서

  - 빅데이터 분석서
